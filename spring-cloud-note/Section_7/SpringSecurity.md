# Spring Security
* Spring Security의 실제적인 구현은 서블릿 필터 (`javax.servlet.Filter` 인터페이스 구현체) 를 통해 이루어짐
* 즉, Spring Security는 필터의 집합체
* Spring Security는 인증과 인가에 관련된 로직을 처리해주는 역할을 함
